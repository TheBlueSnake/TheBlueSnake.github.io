本文主要介绍TheBlueSnake多年OI经验总结的代码规范，有参考$Google$与$Microsoft$的代码规范。可能会随着时间和习惯的改变更新。

概述

代码开头是声明与程序相关注释，随后源码开头是头文件，预编译命令，结构体声明，变量声明，函数的声明及代码，主程序声明及代码。

代码无论是否结束均不可超过100字符，在二元计算符（+，-，*，/，&，|，&&，||，^，>，<，,）处进行分割。且要注意分割时(与)的位置，对于优先级差分较大的代码尽量放在同一行内。

声明，程序注释

声明及程序内容有关的总体注释认为属于代码中，但不属于程序源码内容，全部置于程序代码开头。可以用于声明程序写作时间，作者。编写该程序目的，及该程序实现的目的等。

头文件

头文件置于整个源码的开头，能使用#include<bits/stdc++.h>的时候尽量使用，并将该头文件始终置于代码源码中第一行。其他头文件中，将C++标准库置于其次，C标准库置于再次，交互库置于再次。且能够使用#include<header>时不使用#include "header"。

随后对于C++标准命名空间使用using namespace std;。

预编译命令

多层#if与#endif嵌套时，#endif后应有注释表明其所对应的#if。开头常用的#define定义有如下：

#define INF 0x3f3f3f3f
#define INFll 0x3f3f3f3f3f3f3f3fll

对于其他常量内容定义，应使用const，typedef来定义，如：

const int MAXN = MAX_INT;
typedef long long ll;

结构体声明

结构体的定义只有图论常用边的定义可以使用单行定义，如：

struct edge{int u, v, l; edge(int au = 0, int av = 0, int al = 0){u = au; v = av; l = al;}};

其他非常用结构体无论多简单一律展开书写，具体书写格式参照代码规范细节。

变量声明

变量声明只有图论常用图的定义可以使用单行内混合定义，如：

int tot, frt[MAXN], nxt[MAXM]; edge ed[MAXM];

其余情况下一律单行只有一个变量类型，且相关或在同一代码块中使用的变量定义在相邻几行内。

函数声明

函数使用中不允许使用inline关键字，无论是否是简短函数均不可编写为内联函数。

函数调用时传参应避免传入数组，以及各种巨大的容器，而应传引用。

函数内部代码规范详细参照下方代码规范细节。

主程序声明

主程序声明使用int main()，且必须在程序最后使用返回值return 0;无论该代码是否可能被遇到。

主程序内部代码规范详细参照下方代码规范细节。

代码规范细节

缩进

每一代码块内采用单个Tab缩进，正常使用的IDE应设置TAB宽度为4个空格。

空行与换行

头文件，预编译命令，结构体声明，变量声明，函数的声明及代码，主程序声明及代码之间应该空行，不想关的函数定义两侧应该空行，代码内部的分段（如读入，预处理，处理，输出等）可酌情空行。

行末大括号不换行。能拆开来写的代码尽量拆开来写，消极使用,，积极换行，且在if()，for(; ; )，while()后多使用大括号以便方便进行调试。不主动进行压行。

空格

一定要加空格的地方：

  1. 二元运算符（+，-，*，/，&，|，&&，||，^，>，<）两侧（,除外）。
  2. ,与;的右边（不推荐在同一行内采用;写多条语句）
  3. 三元运算符中?与:的左右两侧。
  4. 指针类型和引用类型的定义中*与&的左侧。（例：int *x;，some_function(int &x)）
  5. 为编译需要必须添加空格的地方。（如不同数据类型之间const long long）
  6. 第5条中特别注意标准库容器套标准库容器里，后方>一定加空格，无论是采用C98编译还是C11或更高编译。（例：vector<pair<int, pair<int, int> > >）

一定不要加空格的地方：

  1. 单目运算符（!，~，&，*，-，++，--）之间
  2. 两个括号之间。（例(variableA * (variableB + (variableC | variableD)))）
  3. 左大括号的左侧。（由不换行策略，做大括号不应处于行首）
  4. 函数名与左括号之间（包括定义，使用）。（例int qpow(int x);）
  5. ,与;的左侧。
  6. *与&的右侧。
  7. .，->，::的两侧。
  8. operator与要重载的运算符之间。（例：operator<）

命名规则

一般使用小写字母命名，使用_进行间隔。不强迫使用英文，可以使用拼音作为变量名（如并查集的fa[]被占用时可使用bing[]）。

特例或注意要点：

  1. 循环变量除数学式中有使用的以外，其余采用i，j，k，p，q等进行循环。
  2. 全局变量采用MAXN作为数量大小，可使用MAXM作为辅助（二维、图、多种输入等），MOD作为取模的量，INF与INFll作为初始最大值。
  3. 模板中可以使用其他模板命名，以及部分算法原名进行大写命名。（如：KMP，FFT，NTT，CDQ）
  4. 临时变量使用tmps，tmp，或者使用下划线开头_val，_ans。
  5. 最大最小值规避使用min，max，使用maximum，minimum或maxi，mini代替。
