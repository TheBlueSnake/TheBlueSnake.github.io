纯数据结构

  1. 基础RMQ/线段操作

    • $O(n\log n)$预处理，$O(\log n)$查询区间操作，支持修改，使用线段树，树状数组维护。

    • $O(n\log n)$预处理，$O(1)$查询RMQ，不支持修改，使用ST表。

  2. 处理一串01串

    • 将01串放在Trie树上，支持$O(\left|S\right|_{max})$批量将大量01串+1，$O(\left|S\right|_{max})$对于每一位内容进行计算。
    • [2020省选联考A卷]树

  3. 处理区间相关操作

    • 与区间方向无关，线段树强行维护，保证复杂度在基础计算后仅增加$O(\log n)$。
    • 会改变区间内方向，如区间取反，使用Splay挤出区间进行维护标记，平摊复杂度也为$O(\log n)$。

图论

  1. 给定一棵树，求树上单独节点内子树相关信息
    • 树上分块，强行利用$O(\sqrt n)$复杂度维护。
    • 使用$O(n)$预处理出树的欧拉（dfs）序，每个点的子树即为一段自区间，使用各种数据结构进行维护计算。
      • [模板]LCA
      • [湖南集训]谈笑风生
  2. 给定一棵树，对于整棵树的形态进行要求判断
    • 进行中序遍历，使用遍历串进行处理，如将对称二叉树变为区间最长回文子串。
      • [NOIp2018普及]对称二叉树

字符串

  1. 对于有限字符串，计算字符间有关的答案（如最长拼接回文串）
    • 利用回文自动机计算。
    • 按照题目要求建图，将字符串问题变为图论上最短/长路，强连通分量，或割点割边等。

奇淫巧技

字符串哈希

  • 只会顺序增加，顺序相关，使用hsh[i] = (hsh[i - 1] * base % MOD + dig[i]) % MOD转变为base进制数进行取模处理，撞哈希可能性较大时可以使用双哈希
  • 需要随机询问，顺序相关，使用同上进制转变法，同时预处理进制$base$次方的$base^i$，对于一段子串$i$至$j$，其哈希函数可以使用(hsh[j] - (hsh[i - 1] * bas[j - i + 1] % MOD) + MOD) % MOD进行计算得出
  • 只会顺序增加，顺序不相关，给每一个元素分配一个随机大权值，若区间内权值和相等，则认为两者是相同的。
  • 需要随机询问，顺序不相关，同上，可以做到随机询问。
  • 选取MOD的时候可以使用unsigned long long自然溢出。

精度误差

  • 对于精度误差会随着计算急速扩大的题目，使用估算并逼近得出项数$n$极大的时候的答案，并反向计算得出最终答案，减小误差影响。

复杂度优化之dp优化

  1. 对于$f[i]=function_{j\leq i}\{f[j]\}$的转移方程
    • 使用线段树将一个$O(n)$变为$O(\log n)$
    • 特殊的，对于形如$f[i]=function_{1\leq j\leq i}\{f[j]\}$的转移方程，可以维护前缀，将一个$O(n)$变为$O(1)$
