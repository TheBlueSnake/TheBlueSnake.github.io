题目描述

Bessie likes downloading games to play on her cell phone, even though she does find the small touch screen rather cumbersome to use with her large hooves.

She is particulatly intrigued by the current game she is playing. The game starts with a sequence of $N$ positive integers ($2\leq N\leq 262,144$), each in the range $1\dots40$. In one moce, Bessie can take two adjacent numbers with equal values and replace then a single number of one greater (e.g., she might replace two adjacent 7s with an 8). The goal is to maximize the value of the largest number present in the sequence at the end of the game. Please help Bessie score as highly as possible.

题目大意

给你一个长度为$N$（$2\leq N\leq 262,144$）的序列，每次操作选两个相邻且相同的数$x$并替换它们为$x+1$，问你最终可以得到的最大数是多少？

输入格式

The first line of input contains $N$, and the next $N$ lines give the sequence of $N$ numbers at the start of the game.

第一行包括一个正整数$N$，随后$N$行每行一个正整数，代表该位置的数字是多少。

输出格式

Please output the largest integer Bessie can generate.

一行，输出你能获得的最大正整数是多少。

输入样例1

4
1
1
1
2

输出样例1

3

考虑用f[i][j]表示从位置i开始合并，那么满足在从i至x - 1的位置可以获得j的最小x。其转移方程类似倍增的形式，可以很容易表示出来

最终答案最大为$40+\log_2n\approx 58$，所以$j\leq 58$即可。代码如下（代码中i和j与题解中的i和j相反）：

#include<bits/stdc++.h>
using namespace std;
#define INF 0x3f3f3f3f
#define INFL 0x3f3f3f3f3f3f3f3fll
const int MAXN = 300100;
typedef long long ll;

int n, arr[MAXN];
int f[60][MAXN];

int main(){int _task = 1; //scanf("%d", &_task);
    while(_task--){
        int ans = 0;
        scanf("%d", &n);
        for(int i = 1; i <= n; i++){
            scanf("%d", &arr[i]);
            f[arr[i]][i] = i + 1;
        }
        for(int i = 1; i <= 58; i++){
            for(int j = 1; j <= n; j++){
                if(!f[i][j]) f[i][j] = f[i - 1][f[i - 1][j]];
                if(f[i][j]) ans = max(ans, i);    
            }
        }
        printf("%d\n", ans);
    }
    return 0;
}
